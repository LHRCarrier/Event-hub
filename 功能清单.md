# EventHub – Community Event Management System

## 功能清单

---

### 1. 系统概述

EventHub 是一个基于 Web 的社区事件管理系统，旨在高效、结构化地管理和组织社区活动。系统采用 Spring MVC + MyBatis-Plus 架构，实现事件生命周期的自动化管理。

**技术栈：**
- 后端：Spring MVC 6.1.6 + MyBatis-Plus 3.5.5
- 前端：JSP + HTML + CSS + JavaScript + AJAX
- 数据库：MySQL 8.0.33
- 服务器：Apache Tomcat
- API文档：Springdoc OpenAPI 2.5.0 + Knife4j 4.4.0
- 认证：JWT (jjwt 0.12.3)

### 3. 系统架构

#### 3.1 项目结构

```
com.bubbles.eventhub/
├── config/                 # 配置层
│   ├── AppConfig.java      # 应用配置（数据源、事务管理）
│   ├── MyBatisConfig.java  # MyBatis-Plus配置
│   ├── OpenApiConfig.java  # OpenAPI/Swagger配置
│   └── WebConfig.java      # Web MVC配置（CORS）
├── controller/              # 控制器层
│   ├── AuthController.java       # 认证接口
│   ├── CategoryController.java   # 分类管理接口
│   ├── DashboardController.java  # 仪表盘接口
│   ├── EventController.java       # 事件管理接口
│   ├── HomeController.java       # 首页接口
│   ├── RegistrationController.java # 注册管理接口
│   ├── SearchController.java      # 搜索接口
│   └── UserController.java        # 用户管理接口
├── dto/                    # 数据传输对象
│   ├── request/            # 请求DTO
│   └── response/          # 响应DTO
├── entity/                 # 实体层
│   ├── Category.java
│   ├── Event.java
│   ├── Registration.java
│   └── User.java
├── exception/             # 异常处理
│   ├── BusinessException.java
│   ├── GlobalExceptionHandler.java
│   └── UnauthorizedException.java
├── filter/                 # 过滤器
│   └── AuthFilter.java     # JWT认证过滤器
├── mapper/                 # MyBatis Mapper层
│   ├── CategoryMapper.java
│   ├── EventMapper.java
│   ├── RegistrationMapper.java
│   └── UserMapper.java
├── service/                # 服务层
│   ├── AuthService.java
│   ├── CategoryService.java
│   ├── DashboardService.java
│   ├── EventService.java
│   ├── RegistrationService.java
│   └── UserService.java
└── util/                   # 工具类
    └── JwtUtil.java        # JWT工具类
```

#### 3.2 核心组件

| 组件类型 | 组件名称 | 说明 |
| :--- | :--- | :--- |
| ORM框架 | MyBatis-Plus 3.5.5 | 继承BaseMapper实现CRUD |
| 分页插件 | PaginationInnerInterceptor | 内置分页支持 |
| API文档 | springdoc-openapi 2.5.0 | 自动生成OpenAPI文档 |
| 增强UI | Knife4j 4.4.0 | 增强型API文档界面 |
| 认证 | JWT (jjwt 0.12.3) | Token认证机制 |
| 参数校验 | Jakarta Validation | 请求参数校验 |
| 日志 | Logback 1.4.8 | 日志配置 |
