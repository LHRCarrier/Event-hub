# EventHub API 接口文档

---

## 1. 文档说明

### 1.1 概述
本文档定义了 EventHub 社区事件管理系统的 RESTful API 接口规范，包含所有业务模块的接口定义。

### 1.2 基础路径
```
http://localhost:8080/eventhub/api
```

### 1.3 数据格式
- 请求格式：`application/json`
- 响应格式：`application/json`

### 1.4 状态码说明

| 状态码 | 描述 |
| :--- | :--- |
| 200 | 请求成功 |
| 201 | 资源创建成功 |
| 400 | 请求参数错误 |
| 401 | 未授权访问 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 2. 认证模块 (Authentication)

### 2.1 用户登录

| 属性 | 值 |
| :--- | :--- |
| **路径** | `/auth/login` |
| **方法** | `POST` |
| **描述** | 用户登录系统 |

**请求体：**
```json
{
  "username": "string (必填，用户名)",
  "password": "string (必填，密码)"
}
```

**成功响应 (200)：**
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "userId": "number (用户ID)",
    "username": "string (用户名)",
    "role": "string (角色：ADMIN/USER)",
    "token": "string (认证令牌)"
  }
}
```

**失败响应 (400)：**
```json
{
  "code": 400,
  "message": "用户名或密码错误"
}
```

### 2.2 用户注册

| 属性 | 值 |
| :--- | :--- |
| **路径** | `/auth/register` |
| **方法** | `POST` |
| **描述** | 新用户注册 |

**请求体：**
```json
{
  "username": "string (必填，用户名)",
  "password": "string (必填，密码)",
  "email": "string (必填，邮箱)",
  "phone": "string (选填，手机号)",
  "realName": "string (选填，真实姓名)"
}
```

**成功响应 (201)：**
```json
{
  "code": 201,
  "message": "注册成功",
  "data": {
    "userId": "number (用户ID)",
    "username": "string (用户名)"
  }
}
```

## 3. 用户管理模块 (User Management)

### 3.1 获取用户列表

| 属性 | 值 |
| :--- | :--- |
| **路径** | `/users` |
| **方法** | `GET` |
| **描述** | 获取用户列表（管理员权限） |
| **权限** | ADMIN |

**请求参数：**

| 参数名 | 类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| page | number | 否 | 页码，默认1 |
| size | number | 否 | 每页数量，默认10 |
| keyword | string | 否 | 搜索关键词（用户名或ID） |

**成功响应 (200)：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "userId": "number (用户ID)",
        "username": "string (用户名)",
        "email": "string (邮箱)",
        "phone": "string (手机号)",
        "realName": "string (真实姓名)",
        "role": "string (角色)",
        "createTime": "string (创建时间)"
      }
    ],
    "total": "number (总数)",
    "page": "number (当前页)",
    "size": "number (每页数量)"
  }
}
```

**成功响应 (200)：**

  "code": 200,
  "message": "success",
  "data": {
    "userId": "number (用户ID)",
    "username": "string (用户名)",
